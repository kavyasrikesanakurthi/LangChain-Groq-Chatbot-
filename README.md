# LangChain-Groq-Chatbot

CLI chatbot built with LangChain and Groq (Llama 3.3 70B) with conversation memory and real-time interaction.

## Features
- Interactive command-line chatbot
- Stores conversation history
- Uses Groq Llama 3.3 70B model
- Environment variable support using dotenv

## Installation

```bash
pip install langchain-groq python-dotenv
```

## Usage

```bash
python chatbotex1.py
```

Type `stop` to end the conversation.

## Technologies Used
- Python
- LangChain
- Groq API
- dotenv
